O site criado é uma página da web institucional dedicada ao Programa Agrinho PR. Ele serve como uma vitrine digital para informar e engajar escolas, professores e alunos sobre a proposta pedagógica do projeto.Tecnicamente, ele é composto por três arquivos interdependentes: o HTML (esqueleto e conteúdo do texto), o CSS (design visual, cores do campo e adaptação para celulares) e o JavaScript (responsável pelas animações de números e abas clicáveis). Na prática, o site funciona de forma leve e rápida, estruturando as informações essenciais sobre sustentabilidade e o concurso anual em um layout moderno.
O site funciona carregando os três arquivos simultaneamente no navegador do usuário. O arquivo HTML exibe as informações sobre o Agrinho de forma estruturada. O CSS entra em ação imediatamente para pintar a página com as cores do campo, organizar o layout e garantir que tudo se adapte perfeitamente à tela de computadores ou celulares.Por fim, o JavaScript ativa as interatividades programadas. Quando o usuário rola a página, os números das estatísticas começam a subir sozinhos; quando clica nas abas do concurso, o conteúdo muda na hora, sem precisar atualizar ou recarregar o site.
